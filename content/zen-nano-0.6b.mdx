`zen-nano-0.6b` is the smallest model in the Zen line, meant for places a GPU is not: a laptop, a phone, a browser tab, an edge box doing autocomplete or intent classification with no network round trip. At this size the interesting constraint is not capability, it is whether the thing loads and answers before a user notices.

The repository carries the transformers weights plus GGUF quantizations at Q4_K_M, Q5_K_M, Q8_0, and f16 for llama.cpp, and an MLX build for Apple silicon, so you are not converting anything yourself.

## Lineage

This is a repackaging of [Qwen/Qwen3-0.6B](https://huggingface.co/Qwen/Qwen3-0.6B) (Apache-2.0) by Alibaba's Qwen team, redistributed under that permissive license for the Zen line. It is worth being exact about this one: unlike the tuned Zen models, nano is a redistribution and a set of format conversions. It was not trained from scratch, and we are not claiming identity training or abliteration on top of it that we did not do. The model you run is Qwen3-0.6B, and every bit of credit for it belongs to Qwen. What we add is the packaging.

`zen3-nano` is the canonical name for this model going forward. New work should start at [zenlm/zen3-nano](https://huggingface.co/zenlm/zen3-nano). This repository stays published and loadable.

## Use it

```python
from transformers import AutoModelForCausalLM, AutoTokenizer

model_id = "zenlm/zen-nano-0.6b"
tok = AutoTokenizer.from_pretrained(model_id)
model = AutoModelForCausalLM.from_pretrained(model_id, dtype="auto", device_map="auto")

messages = [{"role": "user", "content": "Classify this support ticket: cannot reset my password."}]
ids = tok.apply_chat_template(
    messages, add_generation_prompt=True, return_tensors="pt"
).to(model.device)

print(tok.decode(model.generate(ids, max_new_tokens=128)[0][ids.shape[-1]:], skip_special_tokens=True))
```

Or on CPU through llama.cpp:

```bash
hf download zenlm/zen-nano-0.6b gguf/zen-nano-0.6b-Q4_K_M.gguf --local-dir zen-nano
llama-cli -m zen-nano/gguf/zen-nano-0.6b-Q4_K_M.gguf -p "Explain tail latency in one paragraph."
```
