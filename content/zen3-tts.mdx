zen3-tts is the base speech synthesis model in the Zen3 audio line. Give it a short clip of a voice and it will read new text in that voice. It is the right starting point if you are building narration, a character voice, or an agent that should sound like a specific person who consented to it, and it is the checkpoint to fine-tune from if you want your own speaker.

The repository ships two pieces: a talker of 28 layers at hidden size 2048, and a `speech_tokenizer/` module holding the 12 Hz codec. The talker predicts over a 3072-code acoustic vocabulary rather than raw samples, which is what makes the sequence short enough to generate quickly. Total weights are about 1.93B parameters in bf16. A speaker encoder is included, which is the part that reads your reference clip. For preset speakers instead of reference audio see [zen3-tts-custom-voice](/blog/zen3-tts-custom-voice/), and to invent a voice from a written description see [zen3-tts-voice-design](/blog/zen3-tts-voice-design/).

## Use it

```bash
pip install -U qwen-tts
```

```python
import torch
import soundfile as sf
from qwen_tts import Qwen3TTSModel

model = Qwen3TTSModel.from_pretrained(
    "zenlm/zen3-tts",
    device_map="cuda:0",
    dtype=torch.bfloat16,
)

wavs, sr = model.generate_voice_clone(
    text="The build finished in four minutes.",
    language="English",
    ref_audio="reference.wav",
    ref_text="This is the reference recording.",
)
sf.write("out.wav", wavs[0], sr)
```

Clone a voice only with the speaker's consent. Weights: [huggingface.co/zenlm/zen3-tts](https://huggingface.co/zenlm/zen3-tts).
