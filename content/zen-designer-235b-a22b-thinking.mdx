`zen-designer-235b-a22b-thinking` is the reasoning counterpart to [zen-designer-235b-a22b-instruct](https://huggingface.co/zenlm/zen-designer-235b-a22b-instruct). Same sparse mixture-of-experts vision-language stack, tuned to work through a visual problem step by step before committing to an answer.

Reach for it when the argument matters as much as the verdict: comparing two layouts against a constraint, working out why a flow drops users between screens, or auditing a design system for inconsistency across a set of screenshots. For direct answers to direct instructions, the instruct variant is the cheaper call.

## Lineage

This model is a Qwen3-VL-235B-A22B-class model. That much is verifiable from the weights themselves: the architecture is `Qwen3VLForConditionalGeneration`, the tensor layout is Qwen3-VL's, and the parameter total matches Alibaba's Qwen3-VL-235B-A22B release exactly. Credit for the architecture, the pretraining, and the vision stack belongs to the [Qwen team](https://huggingface.co/Qwen) and that line is published under Apache-2.0.

We are not going to name the precise upstream checkpoint here yet. The `base_model` field in this repository currently points at the repository itself, which is not a lineage, and the parameter total is identical across Qwen's Instruct and Thinking releases, so it cannot settle which of the two we started from. Rather than guess and be plausibly wrong, we are confirming it against our training records and will correct the model card when we have it. What Hanzo added is unchanged and is all we ever add: identity training, agentic-data fine-tuning, and abliteration. Nothing here was trained from scratch.

## Use it

```python
from transformers import AutoModelForImageTextToText, AutoProcessor
from PIL import Image

model_id = "zenlm/zen-designer-235b-a22b-thinking"
model = AutoModelForImageTextToText.from_pretrained(
    model_id, dtype="auto", device_map="auto"
)
processor = AutoProcessor.from_pretrained(model_id)

messages = [{
    "role": "user",
    "content": [
        {"type": "image"},
        {"type": "text", "text": "Audit this screen against our spacing scale. Show your reasoning."},
    ],
}]

prompt = processor.apply_chat_template(messages, add_generation_prompt=True)
inputs = processor(
    text=[prompt], images=[Image.open("screen.png").convert("RGB")], return_tensors="pt"
).to(model.device)

out = model.generate(**inputs, max_new_tokens=1024)
print(processor.decode(out[0][inputs["input_ids"].shape[-1]:], skip_special_tokens=True))
```
