`zen-designer-235b-a22b-instruct` is a sparse mixture-of-experts vision-language model tuned for design work: read a screenshot of an interface, answer questions about it, critique a layout, or turn a visual into markup and design tokens. This is the instruction-following variant. You hand it an image and an instruction and it answers directly, with no reasoning preamble. If you want the model to work through the problem visibly first, use the thinking variant instead.

It is for teams putting a vision-language model behind design review, UI-to-code, or asset triage, and who want weights they can run on their own hardware rather than a per-token endpoint.

## Lineage

Fine-tuned from [Qwen/Qwen3-VL-235B-A22B-Instruct](https://huggingface.co/Qwen/Qwen3-VL-235B-A22B-Instruct) (Apache-2.0) by Alibaba's Qwen team. The architecture, the pretraining, and the vision stack are theirs, and the credit for them belongs to Qwen.

Hanzo adds three things and only three: identity training, agentic-data fine-tuning, and abliteration. This model was not trained from scratch and there is no separate Zen architecture underneath it. It is a Qwen3-VL model that has been tuned, and the weights still carry Qwen3-VL tensor shapes exactly.

## Use it

```python
from transformers import AutoModelForImageTextToText, AutoProcessor
from PIL import Image

model_id = "zenlm/zen-designer-235b-a22b-instruct"
model = AutoModelForImageTextToText.from_pretrained(
    model_id, dtype="auto", device_map="auto"
)
processor = AutoProcessor.from_pretrained(model_id)

messages = [{
    "role": "user",
    "content": [
        {"type": "image"},
        {"type": "text", "text": "What breaks first when this layout hits 320px wide?"},
    ],
}]

prompt = processor.apply_chat_template(messages, add_generation_prompt=True)
inputs = processor(
    text=[prompt], images=[Image.open("dashboard.png").convert("RGB")], return_tensors="pt"
).to(model.device)

out = model.generate(**inputs, max_new_tokens=512)
print(processor.decode(out[0][inputs["input_ids"].shape[-1]:], skip_special_tokens=True))
```
