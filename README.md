# Zen LM Blog

Blog for Zen LM -- open-source language models and AI research from Hanzo AI and Zoo Labs Foundation.

**URL**: https://blog.zenlm.org

## Stack

- Next.js 15 + fumadocs
- Tailwind CSS 4
- Static export to Cloudflare Pages

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output is in `out/`.
